StealthChat 🤫
A lightweight, privacy-first chat application designed for users who want to communicate without leaving a digital footprint.
✨ Key Features
Anonymity by Default: No registration required. Users are assigned temporary sessions; no real names or profiles are ever displayed in the chat interface [1, 3].
Secure File Sharing: Integrated support for uploading and sharing files directly within the chat [1].
Zero Logs: Messages are ephemeral and not stored permanently on any server [3, 4].
Responsive Design: Works seamlessly across desktop and mobile browsers.
