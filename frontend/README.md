# StealthChat Frontend

Vue 3 + Vite frontend for StealthChat.

## Stack

- Vue 3
- Vue Router
- Pinia
- Vuetify
- Axios
- Laravel Echo + Pusher protocol client

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
